# Installation: Claude Code Skill

The Docxa Claude Code Skill is the most interactive and guided way to use the platform. It integrates directly into the [Claude Code](https://github.com/anthropic/claude-code) terminal environment.

## Installation

Run the following command in your terminal to install the Docxa skill:

```bash
curl -sSL https://raw.githubusercontent.com/achatt89/docxa/main/install-skill.sh | bash
```

## Usage

Once installed, you can use `/docxa` commands inside your Claude Code session:

- `/docxa init`: Setup a new workspace.
- `/docxa discover`: Analyze the current repository.
- `/docxa interview`: Start a role-aware stakeholder interview.
- `/docxa generate`: Create high-fidelity documentation.

## Why use the Claude Skill?

- **Guided Experience**: Claude helps you answer interview questions and fix missing evidence.
- **Integrated Workflow**: No need to switch between your terminal and an editor.
- **Smart Planning**: Claude automatically suggests the best generation plan based on your repo's state.
